Micello Maps Web API   Private Beta

Welcome to the Micello Maps Javascript Web API, a fully interactive, customizable HTML5 map control that displays Micello indoor maps.

To get started, all you need is a Micello Map API key which can be obtained here. Once you have a key, the following links can help get you started:

  • Tutorial and Programming Guide - Guide to getting started programming the API.
  • Class Reference - JSDoc reference for the public classes in the API.
  • Non-class Reference - This is the format of structures used in the code which are not instantiated as classes. Rather they are constructed by adding properties to a new object.
