Tutorial IV: Overriding Default Behavior

The following three code examples demonstrate overriding default funtions of the map:

Overriding a Mouse Click on the Map

When the map is clicked, the MapCanvas triggers an event which by default is handled by the MapControl object. This map coordinates of the event and the object clicked, if applicable, are passed. The MapControl selects the object and displays any appropriate popup menu or info window. The handler can be overridden to create a custom action. The example below shows an infowindow with the coodinates of the map click and the ID of the object clicked.
