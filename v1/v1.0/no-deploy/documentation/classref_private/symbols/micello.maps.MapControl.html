Class micello.maps.MapControl

This is the basic map control object. The main functionality is controlled by some components of the map. These includes the MapCanvas to display the map, the MapView to handle the viewport to the map, the MapData to handle the actual map data, and the MapGUI to handle the user interactions. Additionally, this class holdds several convenience methods and defines some default behavior.
