Method Detail
addGeometryOverlay(overlay)
This method adds a geometry overlay to the map. A geometry overlay is identical to geometry on the map but is added at run time. The id parameter of the overlay should not be set manually. It will be set automatically by the application.
Parameters:
overlay
addInlay(inlay, asIsolated)
This inserts an inlay object into the the map. An inlay has the same parameters as a geometry overlay except an inlay is used to modify the parameters of an existing geometry object rather than create a new one. The id paramter of the inlay determines which object is modified. Note that no precautions are taken to prevent an inlay from being applied multiple times. If one is, they can be removed in a single call by name or multiple calls of remove by inlay object.
Parameters:
{Inlay} inlay
The inlay to add
{boolean} asIsolated
If the overlay is added to a geometry is, it will be added only to an isolated geometry id if this variable is true. Otherwise, it will be added to all geometry in the group of this geometry. For example, all floors of a departement store.
addMarkerOverlay(overlay, asIsolated)
This method adds a marker overlay to the map. A marker sits above the map and is tied to the map by a single point. The marker does not scale as the map zooms in and out.
Parameters:
{MarkerOverlay} overlay
The marker overlay to add
{boolean} asIsolated
If the overlay is added to a geometry is, it will be added only to an isolated geometry id if this variable is true. Otherwise, it will be added to all geometry in the group of this geometry. For example, all floors of a departement store.
