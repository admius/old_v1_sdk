Class micello.maps.MapCanvas

This class manages rendering the map. It takes source map data from the MapData object, including any annotations (overlays and inlays), and it presents a view of the map as defined from the MapView object. The MapTheme is controlled from the MapCanvas Object. NOTE: Severa constants defined in the constandts file are hardcoded here for performance reasons, rather than using the value assigned to the constant.
