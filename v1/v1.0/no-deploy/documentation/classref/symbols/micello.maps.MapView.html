Class micello.maps.MapView

This class manages the view of the map. It contains functions to set the view as well as to read it.

setZoom(scale, fixedPointX, fixedPointY)
This method set the zoom scale. From within the application, the zoom scale takes on discrete values given by the initial scale of one and the discrete zoom in/out jumps. This function allows arbitrary setting of the zoom scale. This method will not allow a zoom out if the current map size is smaller than the viewport. The fixedPointX and Y are optional arguments. Typically the should be left undefined. If they are set, then that point on the map canavs will remain fix as the zoom occurs.
