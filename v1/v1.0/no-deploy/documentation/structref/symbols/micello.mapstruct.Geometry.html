Field Detail
{integer} cid
This indicates there is an embedded community for this map. The user can navigate inside of the object by opening the community with this ID and the drawing given by the did value.
{integer} did
This indicates there is an embedded map for this community. The user can navigate inside of the object by opening the drawing with this id. If there is also a cid value the community for the map is different from the current community.
{boolean} el
This flag indicates the label is external to the geometry. If this is present, the label geometry should also be used for a hit check.
{integer} gid
This is a groupid id for the geometry. It is true if the geometry is grouped with other geometry. In this case, all geometry in the group should be treated as a common entity, such as a department store that spans multiple levels of a mall. If one of these items is selected or highlighted, then all should be selected/highlighted. If the geometry is not in a group, this variable will likely not be present.
{integer} gt
This is the geometry type, which tells how the shape is used to portray the geometry. For example the shape is a filled area, a linear object or a linear area object where the area is defined by the stroke of the shape. See the constants definitions for allowed values.
{decimal} gw
This variable is used for linear areas. It is the stroke width in map coordinates.
{boolean} i
This flag indicates there is added info associated with this object. If the p flag is not present, info sillbe shown when an item is selected. If the p flag is present, this flag means an info option for the popup will be present.
{integer} id
This is the id for the geometry object.
{String} idat
This is info content. If this is present, this will be used as HTML content for the info page, regardless of whether or not the p flag is present.
{boolean} inl
This flag indicates an embedded drawing can be safely drawn inline with the parent drawing without changing any level structure.
{Array:decimal[5]} l
This is the label area for the object. The ordering of the array is X, Y, Width, Height, Angle in radians.
{String, nominally} lr
This is the label reference. It is nominally a String that tells the text for the label, the name of the icon or the url of the image. In the case where the label is of type geometry, then an explicity geometry value is used.
{integer} lt
This is the label type for the object. See the constant definitions for allowed values.
{integer} mlid
If there is an embedded map and this value is present, the drawing level with this id should be shown when the user opens the drawing. Otherwise, the default level should be choosen.
{MinMax} mm
This is a representation of the bounding box in the format [[xmin,ymin],[xmax,ymax]]. This encompasses the area rendered by the geometry. It may include area for the label. This value will be filled by the application if it is missing when the parameter is first needed. Note that if the geometry or label is updated by an overlay this parameter will not be recalcualted unless it is manually cleared by the overlay.
{String} nm
This is the name to be associated with the object.
{Style} os
This is a style object that should be used to render the object. ("os" = override style)
{boolean} p
This is a flag that indicates a popup should be shown for the object.
{PopupInfo} pdat
This is popup content. If this is present, this data defines the popup format. A popup will be shown regardless of whether or not the I flag is present.
{boolean} r
This flag evaluates to true if this geomtry is the root geometry of the current level.
{shape type specific} shp
This is the coordinates of the object, in a format appropriate for the shape type.
{integer} st
This is the shape type for the object, which tells the type of shape used for this geometry. This determines for format of the shp object. See the constants definitions for allowed values.
{String} t
This is the style that should be used to render the object. (t = "type")
{boolean} u
Unknown flag - this flag indicated the content of this object are unknown. It is a prompt that user input is desired.
